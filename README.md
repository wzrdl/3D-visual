# 3D Model Generator & Local Library

## Project Structure

```
3D-visual/
├── app/
│   ├── __init__.py
│   ├── main_window.py
│   ├── viewer.py
│   ├── api_client.py
│   └── data_manager.py
├── assets/
│   ├── metadata.json
│   └── models/
├── main.py
├── requirements.txt
└── README.md
```

## Running

1. Activate virtual environment:
```bash
source venv/bin/activate
```

2. Run:
```bash
python main.py
```
